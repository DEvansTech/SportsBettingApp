# PSS MVP App

## Getting Started

### Clone this repo
```bash
$ git clone https://github.com/predss/odds-app.git
```

### Install dependencies
```bash
$ cd odds-app
$ yarn install
```

### Run the App on iOS
And then, run the app
```bash
$ yarn ios
```

### Run the App on Android

And then, run the app
```bash
$ yarn android
```
